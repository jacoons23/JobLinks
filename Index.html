Custom Company
Inject_Companies_Heading
This option permits the User to search for a job by Company.
  1. The User selects a Company from the Company List or enters User-Defined Text.
  2. Individual Links appear as buttons for each website.
  3. Each website can be searched without manually opening and searching it.
  4. Click on a Link to open it in another Tab and go there immediately (default action).
  5. On the Generated Links section, move the mouse over the symbol to see the "before" and "after" versions of the Website Address.
  6. Right-Click on a Link and select "Open in another Tab" to open it in another Tab without going there.

     Inject_Tools_Heading
    This option shows various Tools useful to the Developer.
    1. It shows the JobLinks Project History, which details the different main iterations the Project has experienced.
    2. It shows the JobLinks Project Change Log, which details some of the processes completed for the development and testing of the Project.
    3. It shows various CSS elements all in one place, which can be used to verify standardization and integrity of the CSS scripts.
